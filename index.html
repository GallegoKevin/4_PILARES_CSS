4 PILARES CSS

1. Selectores:

Pueden ser:

  • Selectores de etiquetas:

    • Pueden ser: h1, p, div, a, etc.
    Ejemplo:

    HTML:
    <h2>1. Selectores:</h2>
    CSS:
    h2 { color: red; }

  • Selectores de clase:

    • Para agregar o usar una clase en HTML se utiliza el atributo "class" seguido del nombre de la clase.

    Por Ejemplo:

    HTML:
    <p class="example"> HTML </p>

    Para definir la regla de estilo para una clase en css. Primero se debe escribir el hash (#) seguido del nombre que se designo en el atributo "id" en HTML.

    Por Ejemplo:

    CSS:
    .example { background-color: rgb(150, 255, 255); }

  • Selectores de ID:

    • Para agregar un ID a un elemento HTML se utiliza el atributo "id" seguido del "nombre del ID".

    Por Ejemplo:

    HTML:
    <li id="selectorID_third">Selectores de ID</li>

    Para definir la regla de estilo para un ID en css. Primero se debe escribir un punto seguido del nombre de que se designo en el atributo "class" en HTML.

    Por Ejemplo:

    CSS:
    #selectorID_third { background-color: yellow; }

2. Herencia

Proceso por el cual los estilos aplicados a un elemento padre se transmiten a sus elementos hijos.

Por ejemplo:

HTML:
<p>Por ejemplo:</p>
<p>HTML</p>

CSS:
p { color: brown; }
a { color: inherit; }

Todas las etiquetas "p" heredarian el estilo que se le aplico en css y tambien se puede ver que se aplico en css el color "inherit" a la etiqueta "a" para que herede el color de su elemento padre ("p"). Ver más

3. Cascada

Determina qué reglas de estilo se aplican cuando hay múltiples reglas que coinciden con un elemento. Basado en orden, origen y especificidad.

Por ejemplo:

HTML:
<span>Basado en orden, origen y especificidad.</span>

CSS:
span{ color: orangered; }
span { color: purple; }
span{ color: indigo; }

En este caso se aplicaria el valor indigo a la etiqueta span, debido a que aparece en ultimo lugar el codigo.

2. Especificidad

Es un valor que determina qué estilo se aplicará a un elemento cuando hay múltiples reglas que compiten por afectarlo. La especificidad se calcula según el tipo de selector (etiqueta, clase, ID, etc.)

Valor de selectores:

  • Selector de etiqueta: 1
  • Selector de clase: 10
  • Selector dde id: 100
  • Inline (style={propiedad: valor;}): 1000
  • !important : infinito

Por ejemplo:

HTML:
<u class="es_class" id="es_id" style="color:chartreuse">La especificidad se calcula según el tipo de selector (etiqueta, clase, ID, etc.)</u>

CSS:
selector de etiqueta: 1
u{ color: brown; }
!important: infinito
u { color: blue !important; }
selector de clase : 10
.es_class { color: red; }
selector de id: 100
#es_id { color: green; }

En este caso se aplicaria el valor "blue", debido a que !important vale infinito
